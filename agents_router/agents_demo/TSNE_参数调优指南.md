# t-SNE参数调优速查表

## 核心参数（在 draw_t_sne_v2.py 文件顶部修改）

### 1. TSNE_PERPLEXITY（困惑度）
- **作用**：控制关注多少个邻居
- **↑增大**：簇更紧凑，整体更密集
- **↓减小**：簇分得更开，保留更多局部结构
- **推荐值**：30-60（本项目：60）

### 2. TSNE_EARLY_EXAG（早期放大）
- **作用**：早期阶段把簇拉开的力度
- **↑增大**：簇之间距离更大
- **↓减小**：簇更紧凑
- **推荐值**：4-12（本项目：6）

### 3. POINT_SIZE（散点大小）
- **作用**：控制每个点的视觉大小
- **↑增大**：点更明显，适合展示
- **↓减小**：点更精细，适合密集数据
- **推荐值**：30-100（本项目：50）

### 4. FIGSIZE（图尺寸）
- **作用**：画布大小 (宽, 高)
- **↑增大**：有更多空间，标注不重叠
- **推荐值**：(12,10) 或 (14,12)

## 调优策略

**想要簇分得更开？** → 减小PERPLEXITY + 增大EARLY_EXAG  
**想要图更紧凑？** → 增大PERPLEXITY + 减小EARLY_EXAG  
**想要点更大？** → 增大POINT_SIZE
